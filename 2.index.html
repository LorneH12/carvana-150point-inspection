El Estándar Carvana es el punto de referencia para garantizar que cada vehículo sea seguro, limpio, reacondicionado y listo para la entrega. Es la base de cómo Carvana se gana la confianza del cliente. Haga clic en los elementos a continuación para conocer los pilares:

Carvana verifica que cada vehículo cumpla con los requisitos esenciales de seguridad antes de llegar al cliente. Incluye:

  • Brakes
  • Tires
  • Steering
  • Suspension
  • Lights
  • Airbags & onboard diagnostics
  • Frenos
  • Neumáticos
  • Dirección
  • Suspensión
  • Luces
  • Airbags y diagnósticos a bordo
